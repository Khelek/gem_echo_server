# EchoServer

This is a simple echo server on ruby

## Installation

    $ git clone https://github.com/Khelek/gem_echo_server.git

    $ cd gem_echo_server

    $ bundle && rake install
    
To run the test
    
    $ rake test

## Usage

By default server is running on port 8765

    $ echo_server

For use a custom port:
    
    $ echo_server -p 8888
